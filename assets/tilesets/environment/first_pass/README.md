# First-Pass Environment Tileset

This folder is the standardized environment tileset foundation for the current Hobo Survival prototype.

Contents:
- `manifest.json`: machine-readable first-pass asset manifest
- `atlases/`: placeholder atlas blueprints using the approved tile standard

Current standard:
- isometric base tile footprint: `128x64`
- tone: muted, worn, practical, grounded
- scope: camp clearings, forest edges, roadside dirt, and early mud/shore transitions

Use this folder as the stable target for future art generation, manual pixel art, and Godot TileSet assembly.
